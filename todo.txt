📌 *p20 - Adicionar comentarios en celdas*  
Referencia: https://handsontable.com/docs/javascript-data-grid/comments/

En el contextMenu de la Handsontable de citas se debe permitir la opción de agregar comentarios en cualquier celda de las filas que tengan datos así como el mysql necesario 💬.

*Funcionalidad WebSocket:*
Debe operar con los websockets para que cuando un ejecutivo agregue, edite o elimine un comentario, éste se refleje en tiempo real para todos los demás ejecutivos conectados con el elemento cambiado pintado y ligeramente desvanecido para ilustrar el cambio efectuado ✨.

📌 *p21 - Colores de celdas*  
Referencia: https://handsontable.com/docs/javascript-data-grid/formatting-cells/

En el contextMenu de la Handsontable de citas se debe permitir la opción de pintar cualquier celda de filas que tengan datos, así como el mysql necesario 🎨.

*Funcionalidad WebSocket:*
Debe operar con los websockets para que cuando un ejecutivo cambie el color de una celda, éste se refleje en tiempo real para todos los demás ejecutivos conectados con el elemento cambiado pintado y ligeramente desvanecido para ilustrar el cambio efectuado ✨.

*Consideraciones técnicas:*
- Ambas funcionalidades deben integrarse al contextMenu existente
- Sincronización en tiempo real vía WebSocket
- Persistencia en base de datos para mantener estado entre sesiones 🔄
